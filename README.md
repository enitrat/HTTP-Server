# HTTP-Server

## Lancement du serveur
Lancer le serveur en exécutant le fichier .jar présent à la racine
`java -jar HTTP-Server.jar <PORT_NUMBER>`

## Organisation des répertoires
```
project
│   README.md  
│   HTTP-Server.jar
│
└───src
│    │  fichiers .java
│
└───classes
│    │   fichiers .class
│
└───doc
     │  Javadoc API
     │  fichiers      

```
